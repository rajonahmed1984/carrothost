## Image Optimization Tips

1. **Compress all images** before uploading
   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Target: Reduce file size by 50-70% without visible quality loss

2. **Use appropriate formats**:
   - Logos: PNG (transparent background)
   - Photos: JPG (smaller file size)
   - Icons/Simple graphics: SVG (scalable, small size)

3. **Lazy loading**:
   - Images are lazy-loaded automatically by the JavaScript
   - Use `data-src` attribute for lazy loading

4. **Responsive images**:
   - Consider creating @2x versions for retina displays
   - Use responsive image techniques for large images

## Quick Setup

If you don't have images ready, you can:

1. **Use placeholders temporarily**:
   ```
   - Use placeholder.com for testing
   - Example: https://via.placeholder.com/180x45?text=CarrotHost
   ```

2. **Create simple logos**:
   - Use Canva, Figma, or Adobe Express
   - Use brand colors: #f16922 (orange) and #009247 (green)

3. **Download free icons**:
   - Font Awesome (already included)
   - Payment method logos from official brand assets
